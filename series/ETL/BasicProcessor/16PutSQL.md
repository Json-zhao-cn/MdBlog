## **16.PutSQL**
---
### 1.***What is the `PutSQL`?***
1. Executes a SQL UPDATE or INSERT command. 

---
### 2. ***Properties***
1. ***JDBC Connection Pool***：Your DB connection address
2. ***Support Fragmented Transactions***：Sharding transaction with same fragment.identifier.
   - default value: true
3. ***Database Session AutoCommit***：session tranaction
   - default value: true
4. ***Obtain Generated Keys***：If true, any key that is automatically generated by the database will be added to the FlowFile that generated it using the sql.generate.key attribute. This may result in slightly slower performance and is not supported by all databases.
   - default value: false

---
### 3. ***Use case***
1. `ReplaceText` + `PutSQL` to execute the `SQL` Command
2. `ConvertJsontoSQL` + `PutSQL` to execute the `SQL` Command
